# Pixel Wing Duel

A 2-player, same-keyboard, competitive web game.

## How to Play

*   **Player 1:**
    *   `W`: Flap
    *   `S`: Shoot
*   **Player 2:**
    *   `Up Arrow`: Flap
    *   `Down Arrow`: Shoot
*   **Objective:** Shoot the other player to reduce their health to zero. Use power-ups to gain an advantage.

## How to Set Up

Open `index.html` in a web browser.
